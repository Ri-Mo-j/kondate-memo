# 献立メモ（kondate-memo）

家族の献立を整理するためのメモアプリ。**AI駆動開発（AI-driven development）の実践記録**として制作・公開しています。

🔗 **公開URL**：https://ri-mo-j.github.io/kondate-memo/

## 概要

日々の献立を登録・整理し、タグで絞り込めるシンプルな単一ファイルのWebアプリです。
普段はマネジメント／組み込み領域が中心のため、本リポジトリは「AIを用いた開発をどう回すか」を自分の手で検証する場として位置づけています。

## 使用技術

- HTML / JavaScript（単一ファイル構成）
- localStorage によるデータ保持
- GitHub Pages でホスティング

## AI駆動開発で取り組んだこと

- 要件定義から実装・修正まで、AIとの対話を起点に開発サイクルを構築
- 機能ごとにIssue／Pull Requestを切り、AI活用と人のレビューを組み合わせて進行
- 実装例：チップ形式のタグ入力UI、タグのAND条件での絞り込み機能 など
